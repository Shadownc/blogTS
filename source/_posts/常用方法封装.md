##### 修改对象的key值
```JavaScript
//如果data是对象可以免去遍历这一步
replaceKey(data, oldKeyName, newKeyName) {
      data.forEach(item => {
        for (let key in item) {
          if (key == oldKeyName) {
            item[newKeyName] = item[key];
            delete item[key];
          }
        }
      });
      return data;
    }
```
